# OCR Excel Local

App React para convertir texto copiado desde Google Lens en una tabla editable y exportarlo a Excel.

## Requisitos

- Node.js

## Uso

Instala dependencias:

```bash
pnpm install
```

Inicia la app:

```bash
pnpm dev
```

## Flujo

1. Toma la foto de la hoja.
2. Abrela en Google Lens.
3. Copia el texto detectado.
4. Pegalo en la app.
5. Corrige la tabla.
6. Descarga el Excel.

## Nota

La imagen subida en la app solo sirve como referencia visual para corregir mejor los datos antes de exportar.
